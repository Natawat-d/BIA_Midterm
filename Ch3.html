Key idea

Core message: Descriptive analytics answers "what happened?" — but you must first understand the nature of the data (types & measurement scales) and clean it. Statistics summarise the data; visualization communicates it for insight, not pictures.

Nature & taxonomy of data

Data is the lowest level of abstraction from which information and knowledge derive. Quality & integrity are critical — "garbage in, garbage out."

Structured (for computers)Unstructured / semi-structured (for humans)
Categorical: nominal, ordinal · Numerical: interval, ratioText, multimedia (image/audio/video), XML/JSON, HTML, log files

Example: jersey numbers = nominal (labels). Survey rank 1–5 = ordinal (order, but gaps not equal). Temperature in °C = interval (0° isn't "no temperature", so 20° ≠ twice 10°). Weight/kg = ratio (true zero → 20 kg is twice 10 kg). The scale dictates which statistics & chart types are valid.

Data preprocessing — the "art & science"

Real-world data is dirty, misaligned, complex, inaccurate. Four tasks (with feedback loops) make it analytics-ready:

TaskMethods / usage
ConsolidationAccess, select, integrate (SQL, web services, ontology mapping).
CleaningImpute missing values (mean/median/mode/regression/kNN); reduce noise/outliers (binning, IQR/z-score); remove erroneous records.
TransformationNormalize — min-max x'=(x−min)/(max−min) or z-score x'=(x−μ)/σ; discretize/aggregate (binning); construct new attributes.
ReductionReduce attributes (PCA, correlation), reduce records (sampling), balance skewed data (over/under-sample, SMOTE).

Statistical modeling — the measures

Statistics = techniques to characterise & interpret data. Two branches: descriptive (describe the sample) and inferential (conclude about a population from a sample, via confidence intervals & hypothesis tests).

AspectMeasures
Central tendencyMean (outlier-sensitive), median (robust), mode (only valid for nominal)
DispersionRange, IQR = Q3−Q1 (box-plot fences ±1.5·IQR), variance, SD = √variance, quartiles
ShapeHistogram; skewness (asymmetry); kurtosis (peakedness/tails)

Correlation (Pearson r, −1…+1) = strength & direction of a linear relationship — correlation ≠ causation.

Regression — the most widely used technique

  • Simple linear y = β₀ + β₁x; multiple linear y = β₀ + β₁x₁ + … + βₙxₙ.
  • Fit by ordinary least squares (OLS) — minimise the sum of squared errors.
  • Used for explanation (hypothesis testing) and forecasting (prediction).
  • Quality: R² (variance explained), p-values (significance, <0.05), errors MAE/MAD · MSE · RMSE.

Five assumptions (LINEN): Linearity · Independence of errors · Normality of residuals · Equal variance (homoscedasticity) · No multicollinearity.

Logistic regression & time series

Logistic regression — a statistics-based classification algorithm (supervised) with a binary target. Models the log-odds: ln(p/(1−p)) = β₀ + β₁x → p = 1/(1+e−(β₀+β₁x)) (sigmoid, output 0–1); a threshold (e.g. 0.5) turns probability into a class. A frequent exam contrast with linear regression (numeric output).

Time-series forecasting models a series over time (level + trend + seasonality + cyclical + noise) — methods: moving average, exponential smoothing, ARIMA. Regression maps inputs→output; time series maps past→future of one series.

Business reporting & data visualization

Report = Information → Decision. Report types: metric management (KPIs internal / SLAs external), dashboard (many indicators on one page), balanced scorecard.

"The purpose of visualization is insight, not pictures." — Ben Shneiderman.
  • William Playfair invented the line & pie chart (1801); Minard's map of Napoleon's 1812 march = classic multivariate chart.
  • Visual analytics = information visualization + predictive analytics. Leaders: Tableau, Qlik, Microsoft Power BI, SAS.
  • Dashboard layers: Monitoring · Analysis · Management; good dashboards highlight exceptions, allow drill-down, unify systems, need little training.
  • Avoid chart-junk, 3-D distortion, truncated/dual axes, misleading area scaling.

Application cases

Student retention (3.2) — predicting freshman attrition; highlights the data-imbalance problem (an 80/20 set can score 90% accuracy yet 50% precision on the minority class; balancing fixes it); compares ANN, decision tree, SVM, logistic regression, ensembles. NCAA bowl games (3.4) — the same outcome modelled as classification (win/loss) vs regression (point-difference); direct classification with a decision tree scored best (~86%).

Exam focus

Most-tested: measurement scales (interval vs ratio trap); the 4 preprocessing tasks; regression equation, OLS, R²/p-value, the 5 assumptions; logistic regression = binary classification; "insight not pictures"; matching chart to purpose; data-imbalance (accuracy vs precision).
