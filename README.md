# BIA Midterm Prep Hub

Study materials for the **AT82.04 Business Intelligence & Analytics** mid-semester exam (AIT).
Open **[`index.html`](index.html)** for the landing page that links to everything.

## Exam coverage
- **Lectures:** Ch. 1, 3, 4, 5, 6, 9
- **Readings:** R1–R12
- **Format:** **Q1** = Performance Dashboards + Balanced Scorecard applied to AIT (+ sketch the President's dashboard). **Q2** = answer **one of four** readings the instructor selects from R1–R12 (not your own presentation reading, R2).

## How to view
- **Locally:** double-click `index.html` — all files are self-contained HTML (no build step, no dependencies).
- **On the web (GitHub Pages):** push this folder to a repo, then enable *Settings → Pages → Deploy from branch*. The hub will be served at `https://<user>.github.io/<repo>/`.
